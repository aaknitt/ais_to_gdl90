To install rtl-ais:
Follow instructions at:
https://github.com/dgiardini/rtl-ais

$ # Get the source code:
$ git clone https://github.com/dgiardini/rtl-ais
$ # Change to the source dir
$ cd rtl-ais
$ make
$ # Test running the command
$ ./rtl_ais

#use rtl-sdr device index 2, Receive AIS traffic,sent UDP NMEA sentences to 127.0.0.1 port 10110 and log the senteces to console 
./rtl_ais -n -d 2
